أساسيات البرمجة

  • Variable - متغير
  • Function - دالة
  • Loop - حلقة تكرار
  • Condition - شرط
  • Array - مصفوفة
  • Object - كائن
  • Class - فئة (كلاس)
  • Method - طريقة / أسلوب
  • Parameter - معامل (باراميتر)
  • Argument - قيمة مُمرَّرة (أرجومنت)

أنواع البيانات والعمليات

  • Return - أرجع / عودة
  • Boolean - منطقي (صح أو خطأ)
  • String - نص
  • Integer - عدد صحيح
  • Float - عدد عشري
  • Constant - ثابت
  • Scope - نطاق
  • Syntax - الصياغة / بناء الجملة
  • Operator - عامل
  • Expression - تعبير

التحكم والتنفيذ

  • Statement - جملة برمجية
  • Compile - ترجمة (كود → برنامج)
  • Execute - نفّذ
  • Debug - تنقيح الأخطاء
  • Error - خطأ
  • Exception - استثناء
  • Break - إيقاف الحلقة
  • Continue - متابعة الحلقة
  • Initialize - تهيئة
